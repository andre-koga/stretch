# Ease

Mobile-first stretch and meditation PWA. Installable, works offline, no account.

## Stack

- React + Vite + TypeScript
- Tailwind CSS v4
- `vite-plugin-pwa` for install + offline
- Local preferences via `localStorage`
- Procedural ambient audio (Web Audio API)

## Develop

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
pnpm preview
```

## Deploy

Static Vite app — deploy to Vercel. Framework preset: Vite.
